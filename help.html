Features documentation

Raid logistician

Injects clickable cargo fleet composition into spy report to help you build your multiple raid waves. Takes consumption into account.

info_outlineAcronyms are used for better readability

LC stands for Light Cargo (12k cargo)
HC stands for Heavy Cargo (35k cargo)
MBSC stands for Mercury Big Ship Cargo (100k cargo)

Fleet information

Adds some insightful data about currently selected fleet in fleet 1 screen.

info_outline Cargo capacity input can be used in both ways:

file_download When ships are selected it displays the amount of available cargo discarding consumption.

file_upload Resource total can be typed in cargo capacity input and the amount of ships to transport this amount will be adjusted.


Simulator companion

traffic Button to configure fleet as set in simulator.
After clicking on simulator's brand new attack button, you will end up in fleet 1 screen with destination, ship quantity and ship position preset.

Ecology companion

local_florist All the ecology actions automatically clicked just by visiting the ecology page. You are welcome.

Galaxy companion

public Navigation through galaxies and systems using key arrows made possible.

UI sugar

mood Minor touches to make the Space Destiny Design more vivid.

Vote timer

timer Hides the voting banners on homepage and displays the time when you'll be able to vote.

info_outline For now only homepage banner clicks are handled, please do not click on voting banners sent by system messages.
